<img src="PIC2001510952301836.jpg">

### Description

Makes a textbox transparent.

Here's how I do it:

Get the DC from the textbox hWnd.

Paint the textbox text onto the textbox parent.

Paint the parent's image onto the textbox.

Pretty Simple eh?
